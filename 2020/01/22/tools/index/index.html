Vue Mobile UI

名称 链接 介绍
Muse UI https://muse-ui.org/#/zh-CN 属性、参数较多，不简洁。样式风格为 Material Design。
Mand Mobile https://didi.github.io/mand-mobile/#/zh-CN/home 是面向金融场景设计的移动端组件库。
Vant https://youzan.github.io/vant/#/zh-CN/intro 样式较好，组件偏商城。但没有内置表单验证。
Vux https://vux.li 一个凑合的 Vue.js 移动端 UI 组件库。
Mint UI http://mint-ui.github.io 基于 Vue.js 的移动端组件库。

Vue PC UI

名称 链接 介绍
Element https://element.eleme.cn/2.0 Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库。
iView https://www.iviewui.com 一套基于 Vue.js 的高质量 UI 组件库。
Quasar http://www.quasarchs.com 基于 Vue 的 Quasar Framework 介绍 这个框架 UI 组件很全面。
Bootstrap Vue https://bootstrap-vue.js.org Bootstrap Vue 版
Vue Material http://vuematerial.materializecss.cn Vue Material 是一个轻量级的框架， 建立在谷歌的  Material Design  基础上。
